面对市场经济存在的诸多问题，相关各方该怎么办？

市场、政府、税收是什么关系？

这里基于税收和财务的配比关系以及整个经济链，建立函数、模型和体系——包括税收、财务、价值、信息、资信、产品、质量、权责、分配、监管等于一体的体系，提供平台并请各方参与续建，增进透明与公平及效率，减缓信息不对称等问题，扬善惩恶，改进市场和社会秩序。

建立体系的原因—面对市场的失败，相关各方该怎么办？

  • 时代之问

    Q1：在市场经济体制下，如何评价单位、个人、项目、产品的价值与资信？

    Q2：如何抑制贪婪与欺诈、减缓信息不对称等问题？

    Q3：如何帮助人们建立价值与资信品牌，提供营运平台，改进自身、市场与社会发展？

    Q4： 就具体公民而言，国家如何给纳税人一种说法，处理相关责权利关系？

  • 济世之道

    在市场经济的诸方中，税收是基于公用品价值的国之公器，是市场各方财产、价值与权责的分配，是市场诸方关系的交集，能够适度反映并调节与之相应的各方价值和财务及资信状况。税收是经济杠杆、自动稳定器、价值尺度、信息工具、监控与调节手段的统一。应该且能够帮助解决经济与社会等问题。

  • 解决方案

    基数税收和财务关系以及整个经济链，建立一套“与税收和财务相应的财富、价值、信息、资信、诚信、质量、权责等函数体系”——对称性国赋资信体系与平台。诚信而尽责的财赋自有对称性称量与评价，而对避责、造假或无良收支，亦有对称性制衡，进而惩恶扬善，引领价值取向，防范风险与危机，减缓信息不对称等问题，促进市场和社会运转。

体系的发展历程

  • 2007—2008

    经潜心研究数十年，本体系首创于2007—2008年，随后获得广泛认可，拥有资信及相关领域世界唯一的独立自主知识产权，已在中国境内外多领域以多种方式获得快速发展，提供评价标准与服务软件，如服务于一般事主和上市公司的在线软件和微信服务。

  • 2009-2012

    2009—2012年，该体系已经对境内外5万家上市公司并可对数百万家一般企业或个人进行价值与资信评级；并在此基础上提供诸方业务与合作平台。

  • 2013-至今

    境内外相关财经机构和多方人士、中国国家税务总局、银监委、各大银行等，先后应用或力推税务资信。税务资信事业积极推动、参与诚信社会建设，国赋资信事业快速发展。

背景：信息不对称 Asymmetric Information


市场诸方，相互交会而又互守私密、信息不对称。信息不对称，加上资本的贪婪与欺诈的一个风险，就是：各方合作的解体，市场的破产……甚至引发经济危机、社会危机。

三、理论基础 Theory Base

税收价值与信息论

  • 税收价格与价值

    税收是市场各方关系的交集，是国家对财富的分配，是公民因为公用品而向国家付出的价格或贡献，这一价格同时也体现其价值——包括绝对价值、相对价值、增值价值及信息价值等

  • 税收数据关系

    税收数据与计税会计数据之间存在函数与反函数关系，即“税收—兑—财务”关系

  • 税务资信与会计信息

    以上述关系为基础，可建立并分析——“与税收相应的财产价值、财务会计信息”等。

Tax value and information

  • V（财产总价值）= V0(财产的不含税价值——税收相对价值与权责) + Vt(公用品价值——税收绝对价值与权责)（+V△，V0+Vt组合产生的增减价值与社会进退)（+Vi，V0+Vt组合产生的信息价值与相关资信)
  • Vt=F(V)，V=F’(Vt) (税收绝对价值Vt与财产价值V之间存在作用与反作用、函数与反函数关系)
  • 税收与市场及国家的关系——税收是市场各方与公共资源及国家关系的交集，是价值尺度、信息工具、监控与调节手段的统一

四、解决机制 Machine

ISOTAF 对称性国赋资信体系


基于上述背景与理论，可建立一套“与税收和财务相应的财富、价值、信息、资信、诚信、权责等函数体系”——对称性国赋资信体系

在此体系下，诚信而尽责的财赋自有对称性评价，而对避责、造假或无良收支，系统则自动产生自律性和他律性的“对称”反映，进而惩恶扬善。

Function 国赋资信的公式

财赋资信用公式或函数表示为：T(x)*G(y)… E(特别资信)

其中T表示Tax(实缴税收)； x表示税收指标类型；* 表示星级； G表示Grade(等级)；y表示相关经济要素，如r表示收入(revenue)，a表示资产(asset)，c表示资本(capital)，e表示权益(equity)，i表示利息(interest)，L表示负债(liability)，v表示价值(value)等

该体系以国家税收的计征为基础并综合经济链条的诸方信息和管理，以税务稽核为检验，以税票为凭证，以实纳税金为根本——保证和信誉，相对于其他数据或信息，其“含金量”和权威性高。且在确定的税收/财务关系下，如果恶意修改、虚报财务数据或者取得无良收支，系统则自动降低财务状况的评级，甚至还可以依法进行“特别纳税调整”，从而惩恶扬善。
